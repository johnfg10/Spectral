FSpectralModule Class

Performs operations for RGB LED Illumination.

{% highlight cpp %} class FSpectralModule : public IModuleInterface {% endhighlight %}

Methods

CorsairIsEnabled() Checks if Corsair device support was enabled during initialization.
Initialize() Manually initialize the Spectral library. If Spectral is not initialized, this method will be called when trying to perform operations.
IsInitialized() Checks if the Spectral library is initialized.
LogitechIsEnabled() Checks if Logitech device support was enabled during initialization.
SetColor(UCHAR, UCHAR, UCHAR) Sets the color of all LEDs on enabled devices to the specified color.

SetColorForDevice(ESpectralDeviceType, UCHAR, UCHAR, UCHAR) Sets the color of all enabled devices of the specified type to the specified color.
SetColorForDevice(ESpectralDeviceType, FColor) Sets the color of all enabled devices of the specified type to the specified color.
SetColorForLed(ESpectralLedName, UCHAR, UCHAR, UCHAR) Sets the color of the specified LED on enabled devices to the specified color.
SetColorForLed(ESpectralLedName, FColor) Sets the color of the specified LED on enabled devices to the specified color.
SetColorForLeds(int, ESpectralLedName[], UCHAR, UCHAR, UCHAR) Sets the color of the specified LEDs on enabled devices to the specified color.
SetColorForLeds(int, ESpectralLedName[], FColor) Sets the color of the specified LEDs on enabled devices to the specified color.
SetColorForLeds(TArray<ESpectralLedName>, UCHAR, UCHAR, UCHAR) Sets the color of the specified LEDs on enabled devices to the specified color.
SetColorForLeds(TArray<ESpectralLedName>, FColor) Sets the color of the specified LEDs on enabled devices to the specified color.

SetColor(UCHAR, UCHAR, UCHAR)

Sets the color of all LEDs on enabled devices to the specified color.

{% highlight cpp %} static bool SetColor(UCHAR red, UCHAR green, UCHAR blue); {% endhighlight %}

Parameters

red UCHAR

The red channel value for the color to set. On monochromatic devices, only the highest channel value is used.

green UCHAR

The green channel value for the color to set. On monochromatic devices, only the highest channel value is used.

blue UCHAR

The blue channel value for the color to set. On monochromatic devices, only the highest channel value is used.

Returns

bool

True if LEDs on all enabled devices were successfully set.
